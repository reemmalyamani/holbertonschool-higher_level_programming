# SQL More Queries

## Description
This project covers advanced SQL concepts including user privileges, permissions, joins, subqueries, and constraints in MySQL.

## Learning Objectives
- Understand MySQL users and privileges
- Use GRANT and REVOKE statements
- Perform JOIN operations
- Use constraints (PRIMARY KEY, FOREIGN KEY, UNIQUE)
- Work with subqueries

## Requirements
- Ubuntu 20.04 LTS
- MySQL 8.0
- All SQL scripts start with a comment
- SQL keywords in uppercase

## Files

| File | Description |
|------|-------------|
| 0-privileges.sql | Lists privileges of specific users |
