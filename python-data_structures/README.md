# Python - Data Structures

This directory contains introductory Python tasks focused on **lists**, **tuples**, and basic data structure manipulation, as part of the Holberton School Higher Level Programming curriculum.

## Files

* **0-print_list_integer.py**

  * Writes a function that prints all integers of a list, one per line.
  * Uses `str.format()` for output formatting.

## Requirements

* Python 3
* No external modules allowed unless explicitly stated
* Files must start with the correct shebang:

  ```
  #!/usr/bin/python3
  ```
* Code should follow PEP 8 style guidelines

## Usage

Example test file:

```python
#!/usr/bin/python3
print_list_integer = __import__('0-print_list_integer').print_list_integer

my_list = [1, 2, 3, 4, 5]
print_list_integer(my_list)
```

Expected output:

```
1
2
3
4
5
```
