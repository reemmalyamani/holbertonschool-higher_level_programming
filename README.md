# Python - if/else, loops, functions

This project introduces basic control flow and function concepts in Python.
It covers conditional statements, loops, and simple function usage.

## Learning Objectives

At the end of this project, you should be able to explain:

- Why Python programming is awesome
- How to use `if`, `if ... else` statements
- How to use `while` and `for` loops
- How to use `break` and `continue`
- What `pass` does
- How to use `range()`
- What a function is and how to use functions
- What a function returns
- Variable scope in Python
- How to use arithmetic operators

## Requirements

- Allowed editors: `vi`, `vim`, `emacs`
- All files interpreted/compiled on **Ubuntu 20.04 LTS**
- Python version: **Python 3**
- All Python files must:
  - Start with `#!/usr/bin/python3`
  - End with a new line
  - Follow **PEP 8** style (`pycodestyle`)
- All files must be executable
