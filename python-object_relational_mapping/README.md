# Python Object Relational Mapping

## Description
This project demonstrates how to connect Python programs to MySQL databases using MySQLdb and execute SQL queries safely.

## Learning Objectives
- Connect Python to MySQL
- Execute SQL queries using Python
- Use MySQLdb module
- Fetch and display database results
- Follow safe execution practices

## Requirements
- Python 3.8+
- MySQL 8.0
- MySQLdb module installed
