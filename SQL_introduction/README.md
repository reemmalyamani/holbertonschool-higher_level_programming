# SQL Introduction

## Description
This project introduces the fundamentals of SQL and relational databases using MySQL. It covers basic database operations such as creating and deleting databases, listing tables, creating tables, and manipulating data.

The scripts in this project demonstrate how to interact with a MySQL server using SQL commands executed from the command line.

## Learning Objectives
By completing this project, I learned how to:

- Connect to a MySQL server
- Create and delete databases
- List databases and tables
- Create tables with specific columns
- Use SQL scripts to automate database operations
- Understand basic SQL syntax and structure
- Use DDL (Data Definition Language) commands

## Requirements
- Ubuntu 20.04 LTS
- MySQL 8.0
- All scripts executed using:
  ```bash
  cat script.sql | mysql -hlocalhost -uroot -p
