# Python - if/else, loops, functions

This project focuses on learning conditional statements, loops, and basic
function concepts in Python.

## Learning Objectives

By the end of this project, you should be able to explain:

- How to use `if`, `if...else`, and `elif`
- How to use `for` and `while` loops
- How to use `break` and `continue`
- What the `pass` statement does
- How to use `range()`
- What a function is
- How to define and call functions
- What a function returns
- Variable scope in Python

## Requirements

- Allowed editors: `vi`, `vim`, `emacs`
- All files are interpreted on Ubuntu 20.04 LTS
- Python version: Python 3
- All Python files must:
  - Start with `#!/usr/bin/python3`
  - End with a new line
  - Follow PEP 8 style (`pycodestyle`)
  - Be executable

## Files

| File | Description |
|------|-------------|
| `0-positive_or_negative.py` | Prints whether a random number is positive, zero, or negative |
| `1-last_digit.py` | Prints the last digit of a number with conditions |
| `2-print_alphabet.py` | Prints the alphabet in lowercase |
| `3-print_alphabt.py` | Prints the alphabet except `q` and `e` |
| `4-print_hexa.py` | Prints numbers from 0 to 98 in decimal and hexadecimal |
